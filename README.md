1. 复合矩阵中的旋转、平移、缩放要以什么顺序进行，为什么？
顺序应为：先缩放，再旋转，最后平移。因为矩阵乘法不满足交换律，顺序不同会导致结果完全不同。
如果先平移后旋转，物体会绕着世界原点旋转，而不是绕自身旋转，造成“公转”效果而非“自转”。
如果先旋转后缩放，且缩放比例不均匀，物体会沿着旋转后的方向被拉伸，出现歪斜变形。
正确顺序是先缩放改变物体大小，再旋转确定物体朝向，最后平移到目标位置。这样每个变换都作用于物体的局部坐标系，结果最符合直觉。
2. 正交投影和透视投影有什么区别？
核心区别在于是否呈现“近大远小”的效果。
透视投影：模拟人眼或摄像机成像，距离越远的物体在屏幕上看起来越小。平行线在远处会交汇于一点，画面有立体感和纵深感，是3D游戏、虚拟仿真等场景的常用选择。
正交投影：没有近大远小的效果，物体在屏幕上的大小与距离无关。平行线始终保持平行，画面显得“扁平“。
3. 几何变换到屏幕的过程是怎样的？
这个过程通常分为四个主要步骤：
模型变换：将模型从自身的局部坐标系，通过缩放、旋转、平移，放置到世界场景中的指定位置。
视图变换：通过摄像机的位置和朝向，将世界坐标系中的物体变换到摄像机视角下，相当于确定“从哪个角度观察”。
投影变换：根据选择的投影类型（透视或正交），将摄像机视野内的物体映射到一个标准化的裁剪空间中。这一步决定了画面的立体感和可视范围。
视口变换：将裁剪空间中的坐标映射到屏幕的实际像素坐标上，最终生成显示在屏幕上的图像。

首先，ImGui 开始一帧，准备好显示控制面板。控制面板里放了各种滑块和按钮，用来控制四棱台的旋转角度、移动位置、大小缩放、颜色，还有一个下拉框可以选择透视投影还是正交投影。这些控制变量的值会在每一帧之间保留下来，所以滑块拖到哪里，数值就保持在哪里。

然后，程序根据这些控制参数去绘制四棱台。画的时候会根据当前选中的投影类型来计算投影矩阵——透视投影会让物体近大远小，看着有立体感；正交投影则保持物体大小不变，像工程图纸一样，而且可以调整视野大小，数值越大物体看起来越小。

画完四棱台之后，再把 ImGui 的界面画上去，这样控制面板就浮在 3D 场景上面。最后把画好的这帧显示到屏幕上。

用户拖动滑块或者点击按钮时，对应的变量立刻改变，下一帧绘制时就用新值，所以四棱台的位置、大小、颜色、投影效果都会跟着实时变化。点击重置按钮就把所有参数恢复成最初的样子。就这样每一帧重复，实现了实时控制的效果。
